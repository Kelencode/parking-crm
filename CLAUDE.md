# Инструкции для Claude Code

## Перезапуск серверов
- Бэкенд перезапускается АВТОМАТИЧЕСКИ при изменении Python файлов (uvicorn --reload)
- Фронтенд обновляется АВТОМАТИЧЕСКИ через Vite HMR
- После изменений просто скажи пользователю обновить страницу в браузере (F5)
- НЕ НУЖНО просить пользователя перезапускать серверы вручную

## Миграции БД
- При добавлении новых полей всегда добавляй ALTER TABLE миграцию в функцию `_migrate()` в backend/main.py
- Миграция запускается автоматически при старте бэкенда через lifespan
- Новые таблицы создаются автоматически через `Base.metadata.create_all`

## Стек
- Backend: FastAPI + SQLite + SQLAlchemy async (папка backend/)
- Frontend: React + Vite PWA (папка frontend/)
- БД файл: backend/parking.db
